Overview
This Netflix clone is a web application that mimics the user interface of the popular streaming service. Users can browse and view a collection of movies and TV shows. The project utilizes React for the frontend, Tailwind CSS for styling, and Firebase for user authentication.

Features
User Authentication: Users can sign up, log in, and log out to access personalized content.
Movie and TV Show Display: A collection of movies and TV shows is displayed, allowing users to explore and select content.
Responsive Design: The application is designed to be responsive, ensuring a consistent user experience across different devices.
